# POSE

Perform an action.

## Syntax

`pose <action>` `:<action>` `;<action>` (No space)

## Description

Describes your character's action to the room. Using `:` adds a space between
your name and the action. Using `;` appends the action directly to your name.

## Example

`pose waves.` -> "MyName waves." `:waves.` -> "MyName waves." `;'s eyes widen.`
-> "MyName's eyes widen."
