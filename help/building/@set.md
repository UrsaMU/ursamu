# @SET

Sets a flag or attribute on an object.

## Syntax

`@set <object>=<flag>` `@set <object>=!<flag>` (Unset)
`@set <object>/<attribute>=<value>`

## Description

Sets or unsets flags (like WIZARD, STICKY) or sets data attributes on an object.

## Example

`@set me=WIZARD` `@set here=!DARK` `@set here/weather=rainy`
