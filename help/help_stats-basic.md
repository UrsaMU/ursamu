# Basic Stat Usage

## Command Format

`+stat [<player>/]<stat>=<value>`

## Details

- Sets a stat to a specified value
- If no player specified, sets stat for yourself
- Staff can set stats for other players
- To remove a stat: `+stat <stat>=`

## Examples

```
+stat strength=3
+stat dexterity=4
```
