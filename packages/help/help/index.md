# Help

The **help** system aggregates topics from commands, files, and the database.

## Browsing
  `help`                     list all sections
  `help <topic>`             open a topic
  `help <section>/<topic>`   navigate sub-topics

## Sections
  `building`   @commands for builders
  `softcode`   softcode functions and flow control
  `general`    player commands and server info

## For Admins
  `+help/set`     add or update a runtime help entry
  `+help/del`     remove a runtime entry
  `+help/reload`  rescan help files without restarting

## See Also
  +help/set, +help/reload, softcode
