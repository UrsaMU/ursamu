# @describe / @desc

Set the description shown when someone looks at an object.
Alias: `@desc`. Max 4096 characters.

## Syntax
  `@describe <target>=<text>`
  `@desc <target>=<text>`

## Examples
  `@describe here=A cozy tavern with a roaring fireplace.`
  `@desc #12=An old oak door reinforced with iron bands.`
  `@describe me=A weathered adventurer with piercing eyes.`

## See Also
  `@wipe`, `@set`
