# @parent

Set or clear the parent of an object (prototype inheritance).

## Syntax
  `@parent <target>=<parent>`
  `@parent/clear <target>`

## Switches
- `/clear` — remove the parent reference from the target.

## Examples
  `@parent #12=#5`
  `@parent here=#50`
  `@parent/clear #12`

## See Also
  `@set`, `@examine`
