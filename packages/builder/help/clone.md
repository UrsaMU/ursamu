# @clone

Duplicate an object into your inventory. Cannot clone players or rooms.

## Syntax
  `@clone <obj>`
  `@clone <obj>=<new name>`

## Notes
- Copies flags, description, and attributes. Costs 1 quota (staff exempt).
- New name max 200 characters.

## Examples
  `@clone Rusty Sword`
  `@clone #12=Shiny Sword`
  `@clone Torch=Extra Torch`

## See Also
  `@destroy`, `@quota`
