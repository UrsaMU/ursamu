# @examine / @ex

Show detailed object info: flags, owner, lock, location, home, description,
exits, contents, and attributes. Defaults to current room.

## Syntax
  `@examine`
  `@examine <target>`
  `@ex <target>`

## Examples
  `@examine`
  `@examine me`
  `@ex #12`
  `@examine The Tavern`

## See Also
  `@describe`, `@set`, `@lock`
