# @name

Rename an object. Checks for name collisions. Clears a player's moniker.

## Syntax
  `@name <target>=<newname>`

## Notes
- Maximum 200 characters.
- Renaming a player clears their display moniker.

## Examples
  `@name here=The Grand Library`
  `@name #5=Rusty Sword`
  `@name me=Alaric`

## See Also
  `@describe`, `@set`
