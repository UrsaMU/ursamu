combat modifiers  -- Attack pool modifier reference (CoFD core p.92).

Attack pool modifiers (add or subtract dice before rolling):

  Modifier                     Dice Change     Notes
  ---------------------        -----------     ----------------------------
  Aiming (ranged)              +1 per turn     Max +3; lost if Defense
                                               applied. Action costs the
                                               instant slot.
  All-Out Attack               +2              Brawl or Weaponry only.
                                               Lose all Defense for turn.
  Autofire short burst         +1              3 rounds, one target only.
  Autofire medium burst        +2              10 rounds, up to 3 targets.
                                               -1 per target after first.
  Autofire long burst          +3              20 rounds, any number of
                                               targets (ST approval).
                                               -1 per target after first.
  Charge                       --              Move 2x Speed + Brawl/Wpny.
                                               Lose all Defense for turn.
  Offhand attack               -2              Non-dominant hand.
  Pulling a blow               --              Target gains +1 Defense; set
                                               max damage cap in advance.
  Specified target             varies          See: help combat specified.

More:
  help combat range            Range modifiers for ranged attacks.
  help combat conceal-mods     Concealment penalties for the attacker.
  help combat situational      Prone, into-melee, Willpower, surprise.

See also: combat, combat specified, attack, grapple, dodge
