xp costs  -- CoFD 2e Experience cost table and spending rules.

Cost table:
  Trait Type           Cost         Pool
  -------------------  -----------  ----------
  Attribute            4 / dot      Standard
  Skill                2 / dot      Standard
  Skill Specialty      1 flat       Standard
  Merit                1 / dot      Standard
  Willpower (perm)     2 / dot      Standard
  Integrity / morality 2 / dot      Standard
  Supernatural Power   4 / dot      Arcane
  Power Stat           5 / dot      Arcane

Spending rules:
  XP is spent during downtime. Prerequisites must already be met
  (Merit prereqs, Attribute minimums for Skills, template gating for
  supernatural powers).

  Experiences cannot be granted directly; they only flow through the
  Beat economy. Award Beats with +beat/add.

  Arcane-category costs draw from the Arcane pool. Mortals have no
  Arcane pool and cannot purchase Arcane traits.

  Specialty purchases go through +sheet/set specialty/<skill>=<name>.
  +xp/spend does not bill flat-cost categories automatically.

See also: xp, beat, sheet
