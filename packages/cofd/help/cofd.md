cofd  -- Chronicles of Darkness 2e plugin: sheets, chargen, dice, combat,
        beats/XP, conditions/tilts, gear, NPCs, social, and Changeling
        (The Lost) overlay.

QUICK START
  +cg                       Begin or continue character generation.
  +sheet                    View your approved character sheet.
  +roll Strength+Brawl      Roll a CoFD 2e dice pool.

TOPIC GROUPS
  help cofd/character    Sheets, chargen, approval, notes, prove.
  help cofd/dice         Rolls, Beats, XP, extended actions.
  help cofd/status       Health, conditions, tilts, integrity.
  help cofd/combat       Attacks, grapple, dodge, gear, NPCs.
  help cofd/social       Social Maneuvering.
  help cofd/reference    Virtues, Vices, combat merits.
  help cofd/templates    Mortal, Changeling overlay fields.
